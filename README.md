# bopt-slf

A Bayesian optimization library. It features a set level filtration and kernel learning. 

For tutorial notebooks, check out the examples folder.

## Getting started

**Installing with pip**

````
pip install bopt-slf
````

## Dependencies:

* numpy
* scipy
* sympy
* itertools
* pandas
* sklearn
* GPy
* prince
* properscoring
* multiprocess
* matplotlib.pyplot
